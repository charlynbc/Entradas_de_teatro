Hola! Soy el creador de Baco Teatro, un sistema completo de gestión teatral desarrollado con las mejores prácticas y tecnologías modernas.

Sobre el Proyecto

Baco Teatro nació de la necesidad de tener un sistema integral para gestionar todos los aspectos de una compañía teatral:

  • Gestión de grupos teatrales y miembros
  • Control de obras en diferentes etapas
  • Programación y control de ensayos
  • Venta y validación de entradas
  • Reportes financieros y estadísticos

Tecnologías Utilizadas

Backend Node.js + Express
Base de Datos PostgreSQL
Seguridad JWT + Bcrypt
Frontend HTML5 + CSS3 + JavaScript

Características Principales

  • Sistema de Roles: SUPER, ADMIN (Director), VENDEDOR (Actor), INVITADO (Público)
  • Gestión Completa: Desde la creación de un grupo hasta el archivo de obras
  • Control de Ensayos: Programación y estadísticas de asistencias
  • Venta de Entradas: Con códigos QR y validación en tiempo real
  • Reportes: Financieros, de ventas y de asistencias
  • Seguridad: Autenticación robusta y trazabilidad completa
